*** App-Store by HERO.AI ***
A Play Store–style web app built with React.js, featuring app listings, details, and simulated installations — all persisted using Local Storage.

*** Features ***
- Home Page – Highlights featured apps
- All Apps Page – Browse all apps with title, rating, size & downloads
- App Details – View detailed info, charts & ratings
- Installed Apps – Lists all installed apps
- Install/Uninstall Simulation – Real-time updates with Local Storage
- Search & Filter – Quickly find apps
- Error Handling – Graceful fallback for invalid routes/data


React.js * React Router DOM * Local Storage API 
  CSS/Tailwind * AI generated Fake JSON Data
